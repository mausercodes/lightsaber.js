DOCUMENTATION

To use Lightsaber.js download the kit which includes:

  • - Lightsaber Images
  • - Lightsaber Sound Effects
  • - The jQuery Plugin

The Lightsaber can be activated by the click of a button or Konami-code (↑ ↑ ↓ ↓ ← → ← → B A). Default is set to activate on click.

$(function() {

    $(".button").lightsaber();

});

To activate on the click of a button call the lightsaber() function on the desired button.

$(function() {

    $(".body").lightsaber({

        activate: "konami"

    });

});

To activate on Konami-code set the activate property to "konami". The Lightsaber will only activate when you enter ↑ ↑ ↓ ↓ ← → ← → B A on your keyboard.

You can choose what colour you want your Lightsaber to be. The colours available are Red, Green, Blue and Purple. Default is set to Green.

$(function() {

    $(".button").lightsaber({

        color: "blue"

    });

});

To change colours, set the color property to your desired colour. The color value must be entered in lower-case

You can choose not to play Lightsaber sound effects when the Lightsaber is active. Default is set to true.

$(function() {

    $(".button").lightsaber({

        sound: false

    });

});

Set the sound property to false to prevent Lightsaber sound effects from being played.

You can choose if you want Destruction mode active or not. When active, any element with a class of "target" will be destroyed when clicked on with a Lightsaber. Default is set to false.

$(function() {

    $(".button").lightsaber({

        destruction: true

    });

});

Set the destruction property to true to activate destruction mode.

Remember to add the lightsaber.js script in the head of your document.
